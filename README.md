# TÜBİTAK 2204 Proje Sihirbazı

Bu uygulama, TÜBİTAK 2204-A (Lise) ve 2204-B (Ortaokul) yarışmaları için proje raporları oluşturan bir araçtır.

## Özellikler

- ✅ Ortaokul ve Lise seviyesine özel rapor formatları
- ✅ Tek tıkla kapsamlı proje raporu (11 bölüm)
- ✅ TÜBİTAK formatına tam uyumlu
- ✅ İş-zaman çizelgesi (Ocak-Aralık)
- ✅ Danışman öğretmen için sistem yükleme rehberi

## Kullanım

1. Yarışma kategorisini seçin (Ortaokul/Lise)
2. Proje alanını seçin
3. Proje fikrinizi yazın
4. "TAM PROJE RAPORUNU OLUŞTUR" butonuna tıklayın

## Kurulum

```bash
pip install -r requirements.txt
streamlit run app.py
```
